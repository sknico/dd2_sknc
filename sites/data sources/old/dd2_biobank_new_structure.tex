DENNE SIDE SKAL BESKRIVE

\begin{itemize}
\item
  DD2 BIOBANK: infodata om prøverne og kvaliteten af data (IKKE
  resultater)

  \begin{itemize}
  \item
    cpr/projectid (link mellem cpr og projectid)
  \item
    urinprøve, ja/nej
  \item
    blodprøve, ja/nej
  \item
    faste ja/nej
  \item
    vejledato
  \item
    nye data fra biobanken/vejle: antal prøver, glas, osv.
  \end{itemize}
\item
  Biomarker: resultater fra de enkelte analyser (long format)

  \begin{itemize}
  \item
    projectid
  \item
    biomarker (e.g.~c-pep, glucose, il6, \ldots)
  \item
    results (resultatet af analysen)
  \item
    unit
  \item
    study (hvorfor er målingen lavet?)
  \item
    date (f.eks. ved c-pep er der forskel på om analysen er lavet på en
    ny eller gammel prøve, så det kan være relevant at kende datoen for
    analysen så man kan sammenligne den med datoen for blodprøven).
  \end{itemize}
\end{itemize}

Det kan evt. være relevant med et meta-data med henvisning til de
enkelte studier.

Ideelt set vil der for enhver biomarkør være en beskrivelse af
markøren/data/pitfalls/beslutninger/overvejelser etc. Et slags
opslagsværk, så man har samlet information om hver markør (e.g.~c-pep så
man ved hvilke beslutninger der er taget).
